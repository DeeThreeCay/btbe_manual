long-ass serial number

sorry i dont speak in symbols

  • This module has 5 letters with an accent above.
  • Press the left/right arrow to cycle through the letters.
  • For each letter, find its value using the table below.
  • Submit the letters in descending order of value, that is, from highest to lowest value by using the blank submit button.
  • In case of having same values, you may press them in any order.
  • If the value is ≥ 10, change it to its leftmost digit.

*Note: It is possible to have 0 or negative values.

~ - Tilde

Big Talking, Big Explosions (Full Condensed Manual) Accents
1st letter:
If the letter has the diaeresis accent, its value is SN(1st) + SN(last).
Else, if the letter has the tilde accent, its value is SN(1st) × SN(last).
Else, if the letter has the circumflex accent, its value is SN(1st) - SN(last).
Else, its value is SN(1st).
2nd letter:
If the letter is 'î', its value is 4.
Else, if the letter is 'u' with any accent, its value is SN(4th).
Else, if the letter is 'a' with any accent, its value is SN(4th) squared.
Else, its value is 7.
3rd letter:
If the letter is 'ð', its value is 8.
Else, if the letter is 'i' with any accent, its value is SN(7th).
Else, if the letter has the grave accent, its value is 11 - SN(7th).
Else, its value is SN(7th) + SN(5th) - 3.
4th letter:
If the letter is 'y' with any accent, its value is 2.
Else, if the letter is 'e' with any accent, its value is SN(5th) + SN(3rd).
Else, if the letter is 'ü' or 'é', its value is 5.
Else, its value is the number of modules on the bomb.
5th letter:
If the letter is 'å', its value is 1.
Else, if the letter has the tilde accent, its value is SN(2nd).
Else, if the letter is 'o' with any accent, its value is [SN(2nd) + SN(3rd)] × 2.
Else, its value is 4.

Big Talking, Big Explosions (Full Condensed Manual) Memory (Boss Module)

On the Subject of Memory (Boss Module)

Boss Module: a module that provides new information upon solving other modules, and has to be solved at the end (or near the end).

  • The module displays 2 numbers separated by "|".
  • The left number is the stage number, and the right number is the number to be remembered.
  • After solving a module, the display changes and the module advances to a new stage. Remember the displayed numbers in each stage and find a new number (for each stage, separately) via steps stated below.
  • If the number calculated is >10, take the UNIT digit.
  • Input the whole string of numbers calculated to disarm the module.

**Note: Other memory modules, 'Big Button' modules, and Needy modules are ignored.

First input:
If SN(1st) = 3, then the first input is 3.
Otherwise, if SN(1st) < 5, then the first input is stage 1's number + 3. Otherwise, the first input is stage 1's number + 6.
Every other input:
If SN(last) > 5, then the input is the remembered number + SN(last).
Otherwise, the input is the remembered number × SN(last).

On the Subject of the Big Button & Needy Modules

Big Button

Press the button when the bomb time reaches the stated time.

Dual Capacitors (Needy)

Discharge the capacitors occasionally to avoid overloading.

You will receive a strike if either capacitors overload, but both capacitors will then be completely discharged.

Button Masher (Needy)

Press the button the exact amount of times displayed before the time runs out.

Extra presses or insufficient presses will cause a strike.
